개인정보 처리방침

크롬 확장 프로그램 "환율 변환기 (KRW Converter)"의 개인정보 처리 방침입니다.

최종 업데이트: 2025년
요약: 이 확장 프로그램은 어떠한 개인정보도 수집·저장·전송하지 않습니다. 사용자가 드래그한 텍스트는 외부로 전송되지 않으며, 환율 정보를 조회하기 위해 통화 코드(예: "USD")만 환율 API에 전송합니다.

1. 수집하는 정보

본 확장 프로그램은 사용자로부터 다음과 같은 정보를 수집하지 않습니다:

  • 이름, 이메일, 주소 등 개인 식별 정보
  • 브라우징 기록 또는 방문 기록
  • 웹페이지 내용 또는 사용자가 드래그한 텍스트
  • 위치 정보, IP 주소, 기기 정보
  • 쿠키 또는 추적 정보

2. 외부 통신

본 확장 프로그램은 환율 정보를 가져오기 위해 다음 외부 서비스와 통신합니다:

  • open.er-api.com — 무료 환율 API. 통화 코드만 요청에 포함되며, 사용자가 드래그한 텍스트나 식별 정보는 전송되지 않습니다.

환율 데이터는 사용자 기기의 chrome.storage.local에 1시간 동안 캐시되며, 캐시 만료 시 자동으로 새로 가져옵니다.

3. 권한 사용 이유

  • storage — 환율 데이터를 로컬에 캐싱하여 빠른 변환을 제공하기 위함
  • host_permissions: open.er-api.com — 환율 정보 조회
  • 모든 사이트의 콘텐츠 스크립트 — 사용자가 드래그한 텍스트에서 가격을 감지하기 위함. 텍스트는 페이지를 떠나지 않습니다.

4. 데이터 판매 및 공유

본 확장 프로그램은 데이터를 수집하지 않으므로, 어떤 데이터도 판매하거나 제3자와 공유하지 않습니다.

5. 변경 사항

본 방침이 변경될 경우 이 페이지에서 업데이트하며, 새 버전 배포 시 사용자에게 안내합니다.

Privacy Policy

Privacy policy for the Chrome extension "KRW Converter (환율 변환기)".

Last updated: 2025
TL;DR: This extension collects no personal data. Text you select stays on your device. Only the currency code (e.g. "USD") is sent to the exchange rate API — never the selected text, your browsing history, or any identifying information.

1. Information We Collect

This extension does not collect any of the following:

  • Personal identifiers (name, email, address)
  • Browsing history or visited URLs
  • Web page content or text selected by the user
  • Location data, IP address, or device information
  • Cookies or tracking data

2. External Communication

The extension communicates with one external service to fetch exchange rates:

  • open.er-api.com — a free exchange rate API. Only currency codes are included in requests. Your selected text and any identifying information are never transmitted.

Exchange rate data is cached locally on your device via chrome.storage.local for one hour and automatically refreshed when expired.

3. Permission Justification

  • storage — to cache exchange rate data locally for faster conversions.
  • host_permissions: open.er-api.com — to fetch live exchange rates.
  • Content scripts on all sites — to detect prices in the text you select. The text never leaves the page.

4. Data Sale and Sharing

Because the extension collects no data, nothing is sold or shared with third parties.

5. Changes

If this policy changes, we will update this page and notify users via release notes when a new extension version is published.
